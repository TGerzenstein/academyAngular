What is Angular?

Angular is a TypeScript-based web development framework that allows developers to build dynamic and scalable applications.

It is developed by Google and is widely used for creating Single Page Applications (SPA).

Getting Started with Angular

To start working with Angular, you need to install Node.js and the Angular CLI. Follow these steps:

  1. Install Node.js from nodejs.org.
  2. Install Angular CLI using the command: 
    npm install -g @angular/cli
  3. Create a new Angular project: 
    ng new my-angular-app
  4. Navigate to the project folder and start the development server: 
    cd my-angular-app
ng serve

Once the server is running, open http://localhost:4200/ in your browser.
